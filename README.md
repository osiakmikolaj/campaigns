# Campaign Manager

A simple React application for managing marketing campaigns. This application allows users to create, view, edit, and delete campaigns, as well as manage campaign funds using a virtual wallet.

## Features

-   **Dashboard**: View all active and inactive campaigns.
-   **Campaign Management**: Create new campaigns and edit existing ones.
-   **Wallet Integration**: Manage campaign funds with real-time balance updates.
-   **Keyword Selection**: Add keywords to campaigns using a typeahead interface.
-   **Responsive Design**: Built with Bootstrap for a mobile-friendly experience.

## Technologies Used

-   **Frontend**: React.js
-   **Styling**: Bootstrap 5
-   **Backend/Database**: Supabase (accessed via REST API)
-   **Additional Libraries**:
    -   `react-bootstrap-typeahead` for keyword input
